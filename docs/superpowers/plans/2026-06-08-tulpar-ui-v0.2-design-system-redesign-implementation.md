# Tulpar UI v0.2 — Design System Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace v0.1's Tailwind-aligned 6-color palette with a 233-primitive enterprise palette (Navy + Gold brand, 17 chromatic Tailwind-equivalent families, 2 neutrals), switch typography to Source Serif 4 + Source Sans 3, navy-tint shadows, conservative radius. Component API surface stays unchanged.

**Architecture:** Token-only rewrite. Primitives in `packages/tokens/src/primitive/` get new values; `packages/tokens/src/brand/tulpar/light.ts` + `dark.ts` get fully new semantic bindings. Component CSS in `packages/core/src/button/tulpar-button.styles.ts` updates one renamed CSS variable. Consumer apps (`apps/docs`, `apps/playground-ng`, `apps/playground-vue`) get Google Fonts loading + the same CSS variable rename. No component code changes, no new files, no new components.

**Tech Stack:** TypeScript 5.9, Vitest (token tests), @web/test-runner (core tests), Source Serif 4 + Source Sans 3 + JetBrains Mono via Google Fonts CDN.

**Spec:** [`2026-06-08-tulpar-ui-v0.2-design-system-redesign.md`](../specs/2026-06-08-tulpar-ui-v0.2-design-system-redesign.md)
